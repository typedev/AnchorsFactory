# Y positions

Vertical positions can be a font metric (`capHeight`, `xHeight`, `ascender`,
…), another glyph's geometry (`$H`, `$H*5/6`), the glyph's own bounding box
(`box.top`), or the ink itself on a vertical scanline (`outline.top`,
`outline.middle@40`). This chapter shows when to prefer font-wide metrics over
glyph references, and how Y-side `outline` positions mirror the X-side ones
with a *column* as their `@` sample.
