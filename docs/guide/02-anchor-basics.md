# Anchor placement basics

An anchor is a named point written as `name (X Y)` — the parentheses hold one
horizontal and one vertical position, each computed from the glyph's own
geometry rather than typed as raw numbers. This chapter introduces the
placement form, what anchors are for (mark-to-base attachment), and the idea
that the same `frame.position` grammar serves both axes.
