# Getting started

Install `anchorsfactory` from PyPI, point it at a UFO with `--rules default`,
and get a safe `*_anchored.ufo` copy with anchors placed — the source font is
never overwritten unless you ask. This chapter walks through installation, the
CLI's main flags (`--rules`, `--output`, `--in-place`, backups), and writing a
first three-line rules file of your own.
