# Getting started

Install `anchorsfactory` from PyPI, point it at a UFO with a rules file, and
get a safe `*_anchored.ufo` copy with anchors placed — the source font is never
overwritten unless you ask. No rule sets ship with the package: pass a path to
your own `.anchors` file, or a bare set name resolved on `--rules-path` (the
samples to copy live in `examples/rules/`). This chapter walks through
installation, the CLI's main flags (`--rules`, `--rules-path`, `--output`,
`--in-place`, backups), and writing a first three-line rules file of your own.
